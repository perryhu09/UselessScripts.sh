This system is operated by the Bear Cybercommand and runs most of the unit's server operations. Bear Cybercommand has a limited budget due to most of the unit's budget being reallocated to combat operations in the ongoing bear-penguin war, so as a cost saving measure the unit has migrated all of their server operations into this single system. Paular Bear is the Cybercommand member responsible for the administration of this system and has noticed that the system has been compromised by enemy penguin forces. He has asked you for assistance in securing the services across the system and purging the enemy presence in the system.

This system runs both the NGINX and Apache webservers and should only have sites setup to read from the /var/www/html directory. The OpenSSH server is used for remote access and a Samba server is running for the sharing of bear intelligence reports to authorized personnel. Make sure all other Samba shares are removed. There also exists a PostgreSQL database server running that should be secured

Paular Bear has asked that you setup the UFW firewall on the system with rules to allow traffic to the critical services on the system. He has also asked that all traces of penguins and non-work related media and applications to be erased from the system.

In addition to assisting Paular Bear with the security of this system, you have been assigned forensics tasks by the Bear Intelligence Agency to aid them in the investigation of the penguin breach. These tasks can be found on your desktop.
